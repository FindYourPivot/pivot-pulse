# Pivot Pulse — Volume 1, Issue 1

A responsive, maintainable HTML/CSS publication for Pivot Project.

## Files

- `index.html` — publication content
- `assets/css/style.css` — design system and responsive layout

## Editing

Open `index.html` in any text editor. Each publication section is clearly marked with an HTML `<section>` element and an `id`.

Update colours and typography in the `:root` section at the top of `assets/css/style.css`.

## Previewing

Double-click `index.html` or open it in any browser.

## Publishing on GitHub Pages

1. Upload the complete `pivot-pulse-site` folder to a GitHub repository.
2. In GitHub, open **Settings → Pages**.
3. Select **Deploy from a branch**.
4. Choose the branch containing the files and the `/root` folder.
5. Save and wait for the public URL.

## Creating a PDF

Open `index.html` in Chrome or Edge and choose **Print → Save as PDF**. The included print stylesheet removes navigation and optimises spacing.
